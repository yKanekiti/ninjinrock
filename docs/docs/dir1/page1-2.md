## 二項分布

成功確率 $`p`$ のベルヌーイ施行 $`X_1, \ldots, X_n `$ に対して, 確率変数 $`X`$ を

```math
X=X_1 + \ldots + X_n
```

として定義すると, $`X`$ の確率関数は
```math
P(X=k) = \binom{n}{k}p^k(1-p)^{n-k}
```
として与えられる. このときの $`X`$ の分布を二項分布といい,  $`B(n, p)`$ と表す.
また, $`X`$ の期待値および分散は次の通り：
```math
\begin{align*}
E[X] &= np \\
Var[X] &= np(1-p)
\end{align*}
```
